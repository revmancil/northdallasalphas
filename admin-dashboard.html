Event Photo Gallery

Shown on gallery.html

Add photos from chapter events with optional titles and captions. Paste an image URL or upload from your computer (uploads are sent to Supabase Storage bucket chapter-media). To edit the gallery page headline and intro text, open Page Editor and select the Photo Gallery tab.

For large collections, prefer hosted image URLs (e.g. your website or cloud storage) so the gallery stays fast and reliable.

Hosted image URL quick guide
1) Upload images to public storage (Supabase Storage, Cloudinary, or your website).
2) Copy the direct image link and paste it into Photo URL.
3) Confirm the URL opens the image by itself in a browser tab.
Example format: https://<project-ref>.supabase.co/storage/v1/object/public/chapter-media/event-photo.jpg

Brothers Directory

0 records

Maintain your chapter brother directory and mass upload records by CSV. Saving syncs matched records to the member portal directory (matched by email, then Alpha ID). Brothers without an email on file stay in this list only until an email is added.

CSV headers: alpha_id,first_name,middle_name,last_name,suffix,email,phone,status,initiation_chapter,initiation_date,birthday,life_member_date,life_member_number,imdp_certified,risk_mgmt_certified (minimum required: alpha_id). Re-uploading merges by Alpha ID — filled CSV cells update existing rows; empty cells keep current values (no duplicates).
